## Common
The common function calculates the entire grid using the same representation as the example.

We start with a 2000x2000 grid and fill it by iterating over the lines of the input. After some
splitting, we check if we need to add a horizontal or a vertical line, and update the grid
accordingly using `til` to generate the full list of coordinates in the line's direction:
```q
    grid0:{
        a:"J"$first","vs@[;1]"="vs y;b:"J"$".."vs last"="vs y; 
        $[y[0]="x";
            x[b[0]+til 1+b[1]-b[0];a]:"#";
            x[a;b[0]+til 1+b[1]-b[0]]:"#"
        ];
    x}/[2000 2000#".";x];
```

We fill the grid using DFS. Recursion is not a preferred control structure in q due to the limited
stack size and the performance impact, so we have to instead use an iterative implementation with a
stack variable that starts with the origin point of the water:
```q
q)stack:enlist 0 500
```
We iterate as long as there are items on the stack:
```q
    while[0<count stack;
        ..,
    ];
```
We fetch the last (top) value from the stack:
```q
q)curr:last stack
q)curr
0 500
```
We decide what to do in a conditional with several branches. Each branch will be demonstrated with a
different state where the entry condition applies. The branches are evaluated from top to bottom and
the first one to pass is taken, so each condition specified below implicitly assumes that all the
conditions above it are false. For visualization, we look at the current grid and then the same grid
with the current position overwritten with a `"*"` character.

Action 4: this is the most complex one, as it involves filling both left and right, which may result
in either still or flowing water, and also the requirement to recurse in case of a spillover.

We find how far the water can flow on the left by moving through open tiles while the tile
underneath is blocked (still water or wall):
```q
q)leftEnd:curr[1];while[(grid[curr 0;leftEnd-1] in".|")and grid[1+curr 0;leftEnd]in"#~";leftEnd-:1]
q)leftEnd
496
```
We do the same for the right side:
```q
q)rightEnd:curr[1];while[(grid[curr 0;rightEnd+1] in".|")and grid[1+curr 0;rightEnd]in"#~";rightEnd+:1]
q)rightEnd
500
```
